## bookExcerptAPI
A simple API that return excerpts from books. For reading tests purposes.

Built with Node and Express.

### Running Localy

 -- Clone the respository: 
 
 `$ git clone https://github.com/VictorGom3s/bookExcerptAPI` 
 
 -- Install the dependencies: 
 
 `$ npm i`
 
 -- Run the server: 
 
  `$ nodemon app.js`
